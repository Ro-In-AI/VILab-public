librime Compliance Notes ​

  • Component: librime
  • Upstream: https://github.com/rime/librime
  • Version: 1.13.1
  • Source revision: 1c23358157934bd6e6d6981f0c0164f05393b497
  • SPDX: BSD-3-Clause

VILab macOS 输入法仅集成 librime 核心库，并在构建时下载官方发布物用于本地打包。

本仓库不引入以下内容到发布包：

  • Squirrel
  • fcitx5-macos
  • libpinyin
  • 社区词库包，例如 rime-ice

VILab 输入法使用仓库内自带的 vilab_pinyin.schema.yaml 与 vilab_pinyin.dict.yaml。其中 vilab_pinyin.dict.yaml 为 VILab 自有的最小词表，只包含项目自带的基础词汇与产品术语，不引入社区词库包或 GPL/LGPL 前端数据。发布时需一并附带 librime LICENSE 文本。
