VILab ​

VILab 是一个桌面优先的语音平台，也可以向局域网客户端和自托管集成暴露统一的 HTTP 能力层。

这份文档包含什么 ​

  • 如何在本地运行桌面宿主
  • 如何区分客户端与服务端分别负责什么
  • 如何把 SenseVoice 作为服务端 sidecar worker 管理
  • 如何把内嵌 HTTP 服务开放给局域网
  • 如何区分 adminKey 和 external API key
  • 如何调用 /health、/v1/models、/v1/audio/transcriptions
  • 如何理解 raw、smart_clean、polished 这些转录输出变体
  • 如何使用测试平台对比 prompt preset 与 provider/model 效果而不污染历史
  • 如何排查常见的自托管问题

当前产品模型 ​

  • 桌面 App 是 VILab 统一语音层的第一个宿主
  • 内嵌 HTTP 服务是局域网和自托管接入入口
  • 开源版 / 自托管版不要求账号注册或登录
  • 如果未来有官方云服务，那会是另一层产品，不改变自托管模式
