One public entry for docs, downloads, and release trust

This repository is the public face of VILab. The private source repository stays private, while this site handles the parts users need: installers, release notes, public documentation, and the updater feed.

Download with confidence

Every release publishes installers, signatures, updater manifests, and checksums from one stable location.

Ship docs without extra drift

The public site reuses the same docs build that the desktop app serves locally under /docs/.

Keep source control private

Development stays in the private repo. Public users only see the polished distribution surface and changelog.

Built for real product rollout

VILab is more than dictation. It is a reusable voice layer that can power local desktop workflows, LAN services, and future multi-client integrations without forcing one model provider or one deployment shape.

Desktop-first capture

Start with a local host app that handles shortcuts, recording, transcription, cleanup, and operator feedback.

Service-ready architecture

Expose the same core through HTTP endpoints so scripts, LAN clients, and internal tools can reuse it.

Release discipline

Versioned notes, stable update feeds, and public installers make the platform easier to trust and easier to support.

Recent releases

The latest stable versions stay documented here so users can decide when to upgrade and what changed.
