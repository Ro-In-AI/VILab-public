## Quick Install

macOS / Linux / WSL2:

```bash
curl -fsSL https://raw.githubusercontent.com/Ro-In-AI/VILab-public/main/install.sh | bash
```

Windows PowerShell:

```powershell
irm https://raw.githubusercontent.com/Ro-In-AI/VILab-public/main/install.ps1 | iex
```

This branch is synchronized from the private source repository and should not be edited manually.
