VILab 0.2.1

VILab 0.2.1

本次更新

  • 新增热词增强链路，支持本地热词版 SenseVoice、Handy 风格评分和更稳的拼音近似匹配，减少中文专有名词误识别。
  • 接入 MiniMax 转写与改写提供商，并补齐 Anthropic 兼容请求路径，方便在桌面端和 Prompt Lab 中做更多模型对比。
  • 优化历史记录详情页，补充本地音频归档与更多诊断信息，让回看一次识别过程更直观。
  • 修复 Provider 配置持久化与桌面浮层状态同步问题，避免设置保存后丢失或界面状态不同步。
  • 修复 macOS 输入法语音与标点处理问题，提升 IME 录音输入和候选交互的稳定性。

重点验证

  • 热词词典编辑、阈值调节与命中纠正结果
  • MiniMax 提供商接入与 Prompt Lab 对比流程
  • 历史记录详情、音频归档与诊断展示
  • Provider 保存、重启后的配置持久化，以及桌面浮层状态同步
  • macOS 输入法语音输入、标点插入与候选交互

已知说明

  • 热词增强依赖随包提供的 ONNX Runtime 与本地模型资源；首次下载和初始化可能比普通识别慢一些。
  • 当前正式发布仍以 GitHub tag 触发构建和公开仓分发为准，客户端自动更新会跟随公开 Release 刷新。
