VILab 0.2.3

VILab 0.2.3

本次更新

  • 新增历史记录完整导出能力，导出内容包含应用版本、上下文、文本、诊断与相关元数据，便于迁移、排查和归档。
  • 优化 macOS 输入法语音输入悬浮窗与流式输出，支持无输入客户端时的全局语音唤醒，并优先将全局语音快捷键路由到输入法。
  • 统一语音保留、提交与输入法安装清理流程，减少重复输入法实例、旧进程残留和重新安装后的状态不一致。
  • 修复语音唤醒后重复触发导致自动停止，以及输入法快捷键和菜单触发失效的问题。
  • 增强文本上下文测试 playground，补齐 focused text context 场景，方便验证输入法上下文行为。

重点验证

  • 历史记录完整导出，以及导出数据中的应用版本字段
  • macOS 输入法语音悬浮窗、流式输出、保留与提交行为
  • 无输入客户端时的全局语音唤醒，以及全局快捷键路由
  • 输入法安装清理、重复实例处理和旧进程重启流程
  • 诊断、历史记录、overlay 与发布构建相关回归测试

已知说明

  • macOS 与 Windows 发布包仍以 GitHub tag 触发构建，并发布到公开 Release。
  • 如果 macOS 输入法安装后仍显示旧版本，可重新运行安装流程触发清理、重载与 redeploy。
