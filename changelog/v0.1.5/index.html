VILab 0.1.5

VILab 0.1.5

VILab 0.1.5 introduces traceable transcript output variants and a browser-hosted Prompt Lab for internal prompt and model evaluation.

Highlights

  • Session history now keeps multiple transcript outputs such as raw, smart_clean, and polished
  • Regeneration now starts from preserved source outputs instead of repeatedly overwriting the same final text
  • Prompt Lab can compare local Smart Clean runs against cloud Polished runs without polluting normal history
  • Prompt Lab now makes local-versus-cloud execution explicit and supports inline variant renaming
  • Version management now syncs Tauri manifests and docs examples from the package version source

Validation

  • npm run typecheck
  • npm run lab:sync
  • npm test
  • npm run cargo:check
