VILab 0.2.0

VILab 0.2.0

本次更新

  • macOS 输入法升级为双词库 Profile 架构，支持在 官方增强 与 rime-ice（测试项） 之间即时切换。
  • 桌面端设置页与 IME 菜单都新增词库方案入口，切换后会同步写入桥接配置并在输入法侧即时生效。
  • vilab_default 采用官方 rime-luna-pinyin 基础盘、精选 THUOCL 补充词和 VILab 产品术语词库，日常中文输入更稳。
  • rime_ice_test 调整为 no-lua 兼容形态，并补充常用短码 custom_phrase 兜底，方便做真实体验对比。
  • 修复了词库菜单状态与实际运行 profile 不一致、输入法设置跳转错误、配置回写竞态、后台线程输入源状态误报等关键问题。
  • 新增输入法配置指南文档，明确源码配置、安装包资源、运行时 deploy 目录以及本机调优方式。

重点验证

  • macOS 输入法词库切换
  • IME bridge 配置同步
  • rime-ice 短码与全拼候选
  • 打包产物双 Profile 与许可证来源清单

已知说明

  • 当前 vendored librime 仍带有 macOS 15.0 构建 warning；如需更稳妥覆盖 macOS 13/14，建议后续补一次更低 deployment target 的重新构建。
