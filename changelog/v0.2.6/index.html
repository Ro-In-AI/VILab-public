VILab 0.2.6

VILab 0.2.6

Windows 版本

  • 修复 Windows 录音捕获到静音流时仍继续转写的问题：现在会在 STT 前检查音频峰值、RMS 和非零样本数，避免把全 0 音频误识别成短文本。
  • 录音失败提示更明确，会指出当前输入设备没有捕获到有效音频信号，便于切换麦克风或检查系统静音状态。
  • 改进全局快捷键注册失败后的恢复逻辑：当快捷键被其他 VILab 实例或应用占用时，应用会继续运行并重试注册。
  • 加强 Windows 开发启动清理：启动前会清理本项目遗留的 Vite/Tauri 监听进程，减少 5173、9876 端口占用和 app.exe 文件锁导致的启动失败。

验证

  • 通过 TypeScript typecheck、ESLint、Rust 定向回归测试和格式检查。
  • 通过本地 yarn dev 短生命周期启动验证，确认 Vite 与内嵌 HTTP 服务可以正常启动。
  • Windows 安装包发布前会继续执行本地 release build 与 packaged smoke，不使用 GitHub Actions 构建。
