VILab 0.1.6

Highlights

  • Reworked macOS global shortcuts with an event-listener backend so dictation can trigger more reliably on Apple Silicon Macs.
  • Fixed shortcut capture and persistence in Settings so edited shortcuts now save and apply correctly.
  • Improved macOS recording feedback so the overlay reacts more clearly while audio is coming in.

Downloads

  • Apple Silicon macOS: use the .dmg installer attached to this release.
  • Checksums: verify downloads with the attached .sha256 file.

Notes

  • The default macOS dictation shortcut is Cmd+Shift+Option+D.
  • On first use, macOS users must grant Microphone and Accessibility permissions.
  • Version 0.1.6 also updates the Chinese and English quick-start and troubleshooting docs for macOS.
