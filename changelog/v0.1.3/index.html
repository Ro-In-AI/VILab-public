VILab 0.1.3

VILab 0.1.3

VILab 0.1.3 is a Windows hotfix release for startup crashes caused by global shortcut conflicts on user machines.

Fixes

  • Downgraded startup-time global shortcut registration failures from fatal setup errors to runtime warnings.
  • Kept the desktop app running when the default dictation shortcut is already occupied by another Windows application.
  • Recorded the shortcut collision in %APPDATA%\\com.vilab.desktop\\startup-trace.log and host diagnostics so users can still open Settings and change the shortcut.

Documentation

  • Added troubleshooting guidance that asks users to attach %APPDATA%\\com.vilab.desktop\\startup-trace.log to a GitHub issue when the Windows release app still exits during startup.

Release Assets

  • VILab_0.1.3_x64-setup.exe
  • VILab_0.1.3_x64_en-US.msi
  • SHA256 files for both installers

Validation

This release was validated locally before publishing with:

  • Rust regression tests for startup shortcut handling
  • Rust cargo check
  • Full production build
  • Windows release smoke test
