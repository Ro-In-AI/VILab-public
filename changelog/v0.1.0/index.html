VILab 0.1.0

VILab 0.1.0

VILab 0.1.0 是面向中文团队测试与试用的首个 Windows 正式 Release。

Hotfix

  • 2026-04-01：已替换 Windows setup.exe，修复 NSIS 安装包缺少运行时 DLL 导致的启动闪退问题
  • 2026-04-01：修复主窗口关闭后进程未退出，导致再次启动时表现为闪退的问题
  • 2026-04-01：修复 Release 版数据目录迁移与运行时诊断持久化问题，恢复录音记录、日志信息和设置页中的最近一次录音统计展示

本次提供

  • Windows 安装包 VILab_0.1.0_x64-setup.exe
  • Windows 安装包 VILab_0.1.0_x64_en-US.msi
  • 对应安装包的 SHA256 校验文件

本版重点

  • 桌面端语音输入与听写主流程
  • 本地 STT 与云端 STT 的统一路由能力
  • 内嵌 HTTP 服务，便于局域网和自托管方向的后续集成
  • 本地语音模型管理，包括下载、激活与运行时切换
  • 设置、路由、模型状态与录音诊断信息的统一桌面端管理

安装建议

  • 优先使用 VILab_0.1.0_x64-setup.exe，适合大多数 Windows 测试环境
  • 如果你更偏向标准化分发或企业安装流程，可以改用 VILab_0.1.0_x64_en-US.msi

使用前说明

  • 当前 0.1.0 仅提供 Windows 安装包
  • macOS 与 Linux 仍保留为后续支持目标，待原生环境验证稳定后补发对应安装包
  • 普通安装包用户不需要关心 libclang 或 ONNX Runtime 的源码构建依赖
  • 如果要使用本地 STT，请先在应用内下载并激活至少一个本地语音模型

发布与回归

  • Windows Release 现已接入自动化发布工作流：tag 推送后会先执行类型检查、Rust 检查、关键回归测试、正式打包和安装回归
  • 只有全部通过后，GitHub Release 资产才会被上传或替换，避免将明显不可用的安装包发布出去

已知范围

  • 本版重点是首个可下载、可安装、可回归验证的 Windows 正式包
  • 更完整的多平台发布、增量升级与更细粒度的发布门禁会在后续版本继续补齐
