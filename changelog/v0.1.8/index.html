VILab 0.1.8

Highlights

  • Refined the Prompt Lab workspace so source editing, variant controls, and compare actions fit more cleanly within a single screen.
  • Replaced placeholder history content with the user's real recent transcript sessions and limited the rail to the latest 20 entries in a scrollable view.
  • Aligned the Prompt Lab frontend with the latest design file, including denser variant cards, clearer behavior toggles, and updated comparison flow details.

Downloads

  • Windows: use the installer attached to this release.
  • Apple Silicon macOS: use the .dmg installer attached to this release.
  • Public changelog and download pages remain available from the public site.

Notes

  • Prompt Lab now keeps streaming and thinking behavior closer to each variant card so model comparisons are easier to scan.
  • The source workspace removes redundant transcript preview blocks to reduce top-of-screen clutter.
  • Version 0.1.8 focuses on Prompt Lab usability and release metadata alignment rather than backend protocol changes.
