VILab 0.2.2

VILab 0.2.2

本次更新

  • macOS 输入法接入 rime-ice Lua 能力，保留超级简拼与上游 Lua translator，提升拼音、简拼和常用短语候选质量。
  • 候选栏恢复原生 macOS 轻量风格，同时补齐数字选择提示、10 个候选展示、等号/减号与方向键翻页体验。
  • Rime 候选改为按页懒加载，减少候选数量上限导致的假分页问题，并保证数字选词能选中全局候选索引。
  • 修复语音输入、标点提交与光标锚点问题，避免语音文本插到文档开头，或斜杠跑到问号前面。
  • 优化 macOS 打包与安装流程，随包签名 ONNX Runtime 和输入法运行时，安装时自动重启旧输入法进程，降低“装了但还在跑旧版本”的概率。

重点验证

  • rime-ice profile 下的全拼、超级简拼、Lua translator 与 OpenCC 资源加载
  • 候选栏 10 个候选展示、数字选词、等号/减号/PageUp/PageDown/方向键翻页
  • 翻页后跨页候选选择与按页懒加载
  • macOS 输入法语音输入接在已有文本后时的光标位置
  • 问号后继续输入斜杠等标点时的提交顺序
  • macOS debug/release 打包后的输入法 bundle、ONNX Runtime 加载与签名校验

已知说明

  • rime_ice_test 仍作为测试 profile 提供，用于对比更激进的输入体验；如果遇到词库部署缓存异常，可重新安装输入法触发清理与 redeploy。
  • macOS 发布包仍以 GitHub tag 触发构建和公开仓分发为准，客户端自动更新会跟随公开 Release 刷新。
